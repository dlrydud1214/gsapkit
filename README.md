# GSAPKit

GSAP 애니메이션을 화면에서 조절하고 Vanilla JS 또는 React 코드로 복사하는 비주얼 빌더 MVP입니다.

## 실행

```bash
npm install
npm run dev
```

## 현재 기능

- Fade Up, Scale In, Slide Left, Stagger Cards 프리셋
- duration, distance, stagger, ease 조절
- ScrollTrigger 코드 포함 여부 선택
- 미리보기 다시 재생
- Vanilla JS / React 코드 생성 및 복사
- 반응형 UI
